# 💊 Smart Pill Reminder System for Elderly Patients

An Android application developed to help elderly patients manage their medicines and doctor appointments efficiently. The application sends timely medicine reminders, stores patient records securely, and provides AI-powered health assistance.

---

## 📌 Problem Statement

Many elderly patients:

- Forget to take medicines on time.
- Miss doctor appointments.
- Find it difficult to maintain medical records.
- Need quick answers to health-related questions.

This application solves these problems by providing medicine reminders, appointment tracking, and AI-based health support.

---

## ✨ Features

✅ User Registration and Login

✅ Secure Authentication using Firebase

✅ Medicine Reminder with Notifications

✅ Doctor Appointment Scheduling

✅ Calendar Integration

✅ AI Health Assistant using Gemini API

✅ Secure Data Storage using Firebase

## 📱 Application Workflow

1. User registers or logs in.
2. User adds medicine details.
3. Medicine information is stored in Firebase.
4. Alarm Manager schedules reminders.
5. Notification appears at the specified time.
6. User schedules doctor appointments.
7. Gemini AI answers health-related queries.
